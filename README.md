# Игра: Путь Судьбы
## Описание проекта
Консольная RPG-игра, разработанная на Python, с тремя сюжетными ветками, системой артефактов и пошаговой боевой системой. Игра реализует все требования задания: три сюжетные линии, систему сохранения прогресса, копилку артефактов и интерактивные выборы, влияющие на развитие сюжета.
## Особенности игры
1. Три сюжетные ветки

Путь Воина - фокус на физической силе и чести

Путь Мудреца - акцент на магии и знаниях

Путь Странника - свобода и приключения


2. Система сохранения прогресса
   
Автоматическое сохранение персонажа в JSON-формате

Пароли хранятся в открытом виде (по требованию задания)

Возможность загрузки предыдущих сохранений

3. Копилка артефактов

Коллекционирование уникальных артефактов

Возврат артефактов в копилку при отказе от сохранения

Автоматическая генерация новых артефактов при сборе всех существующих


4. Боевая система

Пошаговые бои на основе классов из исходного кода

Система характеристик (сила, ловкость, интеллект)

Боссы с двумя фазами боя


5. Система выбора

Ветвящиеся сюжетные линии

Каждый выбор влияет на развитие истории

История всех выборов сохраняется

## Структура проекта 

game.py - Основной файл игры

player.py - Класс игрока и система сохранения

kopilka.py - Система артефактов

users.json - База данных пользователей (автосоздание)

artifacts.json - База данных артефактов (автосоздание)

saves/ - Папка с сохранениями (автосоздание)

requirements.txt - Файл зависимостей

README.md - Документация
